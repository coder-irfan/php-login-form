# PHP Login & Registration System

A simple and secure PHP login and registration system with:

- User registration
- Password hashing
- Login & logout
- Session-based dashboard
- Form validation (server-side)
- Styled using CSS

---

## Features

- Registration with email & username uniqueness
- Passwords stored securely with `password_hash()`
- Login system with session management
- Dashboard page protected from unauthorized access
- Logout system destroys session completely
- Clean, modern CSS styling
